# Privacy Policy — Pomodof

**Last updated:** April 27, 2026

## Overview

Pomodof is a menu bar Pomodoro timer for macOS. Your privacy is important to us. This policy explains what data the app accesses and how it is handled.

## Data Collection

Pomodof does **not** collect, store, or transmit any personal data to external servers.

## Apple Reminders

Pomodof can access your Apple Reminders to display tasks and log completed sessions. This data:

- Is accessed **only on your device** using Apple's EventKit framework
- Is **never sent** to any server or third party
- Is only modified when you explicitly start or complete a session

You can deny Reminders access and still use the timer without task integration.

## Network Access

Pomodof connects to the GitHub API (`api.github.com`) **only** to check for new versions, and only when you enable "Check for Updates" in settings. No personal data is sent in this request.

## Local Storage

Pomodof stores your preferences (timer durations, settings) locally using macOS UserDefaults. This data never leaves your device.

## Analytics & Tracking

Pomodof does **not** include any analytics, tracking, crash reporting, or advertising frameworks.
